   images      -   Holds all the images
   inc         -   Holds include files
   libraries   -   Holds libraries used on the site

   main.css  -  Main stylesheet


Images Directory File(s):

Inc Directory File(s):
   template.php  -  Template file


Inc Directory File(s):
   Page.php  -  Holds the Page class


Root Directory File(s):
   about.php      -   About page
   contact.php    -   Contact us page
   index.php      -   Home page
   prototype.php  -   Contains the components that every page will have so that you can create a new page quickly.
   services.php   -   Services page
